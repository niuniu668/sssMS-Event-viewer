# Qt Waveform Viewer v4.0.0

发布日期：2026-05-05

## 主要更新
- Data View 新增“复制名称”和“在资源管理器中打开”的右击菜单，Event/Data/Pick 三类目录与文件列表均可用。
- 启动和关闭时会自动保存会话状态，包括上次打开的文件夹、最后选中的文件、当前页签和常用勾选项。
- Data View 支持在“全波形 / 切片”之间切换，切片按事件时间优先截取 4000 点窗口。
- 文本型 data 文件自动识别逻辑增强，未知后缀但内容像文本波形的数据会优先按文本解析，避免误判成二进制。
- Pick View 的波形显示与切片逻辑进一步整理，切片窗口长度与 marker 偏移更稳定。

## 兼容性说明
- 原始 data 文件仍支持直接打开，无需手动改后缀为 csv。
- 目录右击菜单会根据条目类型自动执行对应操作：文件定位到资源管理器中的文件，文件夹直接打开文件夹。
